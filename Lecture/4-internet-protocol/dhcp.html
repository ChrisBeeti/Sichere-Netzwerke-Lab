DHCP
Dynamic Host Configuration Protocol — IP-Adresszuweisung & DORA-Prozess
DORA-Prozess — Nachricht wählen oder vollständig animieren
Bereit
DORA-Sequenz
Übertragene Parameter
Der DHCP-Server teilt dem Client bei ACK mit:
  • IP-Adresse (Leih-Adresse)
  • Subnetzmaske
  • Default-Router (Gateway)
  • DNS-Server
  • Lease-Zeit (Gültigkeitsdauer)
Ports
Server
UDP 67
Client
UDP 68
IPv6 Server
UDP 547
IPv6 Client
UDP 546
Vorteile
✓ Automatische IP-Konfiguration
✓ Weniger Fehlerquellen (kein manuelles Tippen)
✓ Zentrale Verwaltung aller Netzwerkparameter
✓ Flexible Adressänderungen ohne Clienteingriff
Nachteile
✗ Zusätzlicher Netzwerkverkehr (Broadcasts)
✗ DHCP-Server als Single Point of Failure
✗ Administrationsaufwand für Reservierungen
